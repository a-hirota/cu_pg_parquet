## メモリ効率改善

```
従来版: 218MB × 2回 = 436MB メモリアクセス
統合版: 218MB × 1回 = 218MB メモリアクセス

効率改善: 50%削減
実行時間: 26.3%短縮
```

## データ構造

```
PostgreSQL Binary Format:
┌──────────────────────────────────────┐
│ Header (19B) │ Row1 │ Row2 │ ... │ RowN │
└──────────────────────────────────────┘

Row Structure:
┌─────────────────────────────────────────────┐
│ ncols(2B) │ field1_len(4B) │ field1_data │ ... │
└─────────────────────────────────────────────┘

GPU Output:
field_offsets[row][col] → データ開始位置
field_lengths[row][col] → データ長（-1=NULL）
```
